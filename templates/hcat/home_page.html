Tuatara: Project Data Tracking

About

Tuatara is a database to keep track of projects that the HCA is integrating or interested in integrating into the Atlas. It can answer data roadmap questions like "What is the current status of single nuclei-seq? I'd like to know when we can expect it so I can prioritize pipeline development, and from who so I can read papers from the labs." Tuatara is built on top of Python, Django, and a relational database that we hope will replace several spreadsheets.

Access

If you are interested in tracking data with Tuatara through a prototypical web interface, please email clayfischer@ucsc.edu. We will be rolling out broader access as we iterate through feedback.
