API

A REST-ful API allows one to query for current status of projects. Combined with the tracker.data.humancellatlas.org API, a full dashboard can be built to show all data known and in progress. Not all data is available through the API due to GDPR constraints, but enough data to make informed decisions based on known projects is available. Here's a list of the restful urls supported by the API:

  • https://tuatara.data.humancellatlas.org/api - lists all classes and the number of objects in each class
  • https://tuatara.data.humancellatlas.org/api/project - lists information on all projects
  • https://tuatara.data.humancellatlas.org/api/project/[project short name] - list info on project with short name
  • https://tuatara.data.humancellatlas.org/api/projectstatus - List and describe values in project.status field
  • https://tuatara.data.humancellatlas.org/api/contributor - List contributors to any project
  • https://tuatara.data.humancellatlas.org/api/tracker - Post-submission tracking of project
  • https://tuatara.data.humancellatlas.org/api/cdnalibraryprep - Types of single cell RNA sequencing assay
  • https://tuatara.data.humancellatlas.org/api/disease - Diseases currently in any project.
  • https://tuatara.data.humancellatlas.org/api/organ - Organs currently used in any project.
